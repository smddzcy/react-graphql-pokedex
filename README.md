# React & GraphQL Pokedex

A fun project to learn GraphQL and how it integrates with React.

### Installation

Please follow the following steps to get up and running.

  1. Install dependencies:

    ```javascript
    npm install && flow-typed install
    ```

  2. Run the unit tests

    ```javascript
    npm test
    ```

  3. Run the e2e tests

    ```javascript
    npm run nightwatch
    ```

  4. Run the static type checker

    ```javascript
    npm run flow
    ```

  5. Start the webpack server

    ```javascript
    npm run server
    ```
